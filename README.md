# 📺 JavaScript Playlist

Welcome to the **JavaScript Playlist** – a curated collection of JavaScript examples, exercises, and mini-projects aimed at helping beginners to intermediate developers learn JavaScript in a structured, hands-on way.

---

## 📌 About This Repository

This repository includes:
- 🧠 **Basic to advanced JS concepts**
- ⚙️ Hands-on **code snippets** for practice
- 🛠️ Small **projects** to reinforce learning
- 📂 Well-organized folder structure for easy navigation

Whether you're brushing up on fundamentals or diving into more complex features, this playlist will guide you step-by-step.

---

## 📂 Folder Structure

```bash
javascript-playlist/
├── 01_basics/
├── 02_dom_manipulation/
├── 03_events/
├── 04_async_js/
├── 05_projects/
└── README.md
````

---

## 🚀 Topics Covered

* ✅ Variables, Data Types, Operators
* ✅ Functions and Scope
* ✅ Arrays and Objects
* ✅ DOM Manipulation
* ✅ Event Handling
* ✅ Promises, async/await
* ✅ LocalStorage & SessionStorage
* ✅ Mini Projects (e.g., Calculator, Todo App)

---

## 🛠️ Requirements

To run the examples:

* 📦 [Node.js](https://nodejs.org) (for some backend-based exercises)
* 💻 Any modern browser (for frontend projects)
* 🧰 A code editor like [VS Code](https://code.visualstudio.com/)

---

## ▶️ How to Run

### Run in Browser:

Just open the `.html` files in your browser.

### Run with Node.js:

```bash
node your-file.js
```
